# Exercise 2.1 – Bootstrap Directory Structure

## CSS Folder
- Contains Bootstrap CSS files.
- Used to style web pages.
- Important file: `bootstrap.min.css`

## JS Folder
- Contains Bootstrap JavaScript files.
- Used for interactive components like Modals, Dropdowns, Accordions, and Carousels.
- Important file: `bootstrap.bundle.min.js`

## Icons Folder
- Bootstrap Icons are provided as a separate package.
- Used to add icons like Home, Search, User, Bell, etc.
